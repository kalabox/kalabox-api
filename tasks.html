tasks

Module for managing Kalabox tasks.

Methods

<inner> getCount() → {number}

Returns the count of registered tasks.
Returns:
- The count of registered tasks.
Type
number

<inner> registerTask(names, task)

Registers a task that later can be run by Kalabox.
Parameters:
Name Type Description
names string.<array> Array of names representing a map to the task.
task function Function to run as the task.
Properties
Name Type Description
next function Callback to be called when the task is complete.
Example
kbox.core.tasks.registerTask([app.name, 'name'], function(next) {
  console.log('app name: ' + app.name);
  next(); // Must be called to return control back to the Kalabox.
});
