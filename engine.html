Module: engine

engine

Module for interfacing with the Kalabox container system implementation, and the provider of the container system. NOTE: The container system and the provider are not always the same. For example using docker on a Mac would use docker as the container system, however it would be provided by boot2docker.

Members

<inner> up

Gets the engine provider into the state it needs to be in to do engine things.
Example
kbox.engine.up(3, function(err) {
  if (err) {
    done(err);
  } else {
    var app = kbox.core.deps.lookup('app');
    kbox.app.rebuild(app, done);
  }
});
