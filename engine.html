Module: engine

engine

Module for interfacing with the Kalabox container system implementation, and the provider of the container system. NOTE: The container system and the provider are not always the same. For example using docker on a Mac would use docker as the container system, however it would be provided by boot2docker.

<static> create(createOptions, callback)

Creates a container.
Parameters:
Name Type Description
createOptions object [optional] - Object containing the create options for the container.
callback function Callback called when the container has been created.
Properties
Name Type Description
error error
data object Object containing the created containers information.
Example
engine.create(component.installOptions, function(err, container) {
  if (err) {
   callback(err);
  } else {
    if (container) {
      component.containerId = container.cid;
      fs.writeFileSync(
        path.resolve(component.containerIdFile), container.cid
      );
   }
    events.emit('post-install-component', component, function(err) {
      callback(err);
    });
  }
});

    stream.on('error', function(err) {

  }
});
