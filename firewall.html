Module: firewall

firewall

Kalabox firewall utility module.

Methods

<static> isBlockingAll(callback)

Gets status of firewall blocking everything. **ONLY WORKS ON OSX**
Parameters:
Name Type Description
callback function Callback function.
Properties
Name Type Description
err error Possible error object.
result boolean Boolean result set to true if firewall is blocking everything.
Example
kbox.util.firewall.isBlockingAll(function(err, isBlockingAll) {
  if (err) {
    throw err;
  } else if (isBlockingAll) {
    throw new Error('Firewall is blocking everything!');
  }
});

<static> isOkay(callback)

Gets status of firewall being in an okay state. **ONLY WORKS ON OSX**
Parameters:
Name Type Description
callback function Callback function.
Properties
Name Type Description
err error Possible error object.
isOkay boolean Boolean result set to true if firewall is in an okay state.
Example
kbox.util.firewall.isOkay(function(err, isOkay) {
  if (err) {
    return throw err;
  }

});
