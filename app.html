Module: app

app

Module for interacting with and managing user defined apps.

Methods

<static> create(name, config, callback)

Creates an app.
Parameters:
Name Type Description
name string The name of the app.
config object An app config object.
callback function Callback function called when the app is created
Properties
Name Type Description
error error An error if any.
app object The instantiated app object.
Properties:
Name Type Description
name string String name of the application.
domain string The domain name. -> appName.kbox
url string URL of the app.
dataContainerName string Name of the data component for the app.
appRoot string Root directory for the app.
config object The app's config object.
plugins object.<array> List of the app's plugins.
components object.<array> List of the app's components.
Example
kbox.app.create(config.appName, config, function(err, app) {
 callback(err, app);
});

<static> get(appName, callback)

Get an app object from an app's name.
Parameters:
Name Type Description
appName string App name for the app you want to get.
callback function Callback function called when the app is got.
Properties
Name Type Description
error error Error from getting the app.
app app App object.
Example
kbox.app.get('myapp', function(err, app) {
  if (err) {
    throw err;
  } else {
    console.log(app.config);
  }
});

<static> list(callback)

Lists all the users apps known to Kalabox.
Parameters:
Name Type Description
callback function Callback called when query for apps is complete.
Properties
Name Type Description
error error
apps app.<array> Array of apps.
Example
kbox.app.list(function(err, apps) {
  if (err) {
    throw err;
  } else {
    apps.forEach(function(app) {
      console.log(app.name);
    });
  }
});

<static> rebuild(app, callback)

Rebuilds an apps containers. This will stop an app's containers, remove all of them except the data container, pull down the latest versions of the containers as specified in the app's kalabox.json. You will need to run `kbox start` when done to restart your app.
Parameters:
Name Type Description
app object App object you want to rebuild.
callback function Callback called when the app has been rebuilt.
Properties
Name Type Description
errors array Error from rebuilding components.
Example
kbox.app.rebuild(app, function(errs) {
  if (err) {
    throw err;
  } else {
    console.log('App rebuilt!.');
  }
});
